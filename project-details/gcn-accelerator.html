ASIC Acceleration for Graph Convolutional Neural Networks (RTL to GDSII)

Arizona State University
Mar 2024 - May 2024
Tempe, Arizona
SystemVerilog Cadence Innovus ASIC RTL Design

Project Overview

This project involved designing and implementing RTL for a Graph Convolutional Network (GCN) accelerator in System Verilog, demonstrating in-depth understanding of graphics processing architecture. The project completed the full RTL to GDSII flow using industry-standard EDA tools.

The GCN accelerator was synthesized into a gate-level netlist and underwent complete physical design using Cadence Innovus, including floor-planning, placement, routing, and timing closure to achieve optimal area utilization.

Key Achievements

  • Designed and implemented RTL for Graph Convolutional Network accelerator in System Verilog
  • Demonstrated in-depth understanding of graphics processing architecture
  • Synthesized RTL into gate-level netlist achieving 15% reduction in area utilization
  • Performed complete physical design using Cadence Innovus
  • Completed floor-planning, placement, routing, and timing closure
  • Successfully completed full RTL to GDSII design flow
