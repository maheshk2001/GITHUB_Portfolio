Implement the GSelect Branch Predictor

Arizona State University
Aug 2024 - Oct 2024
Tempe, Arizona
C++ Gem5 Branch Prediction Computer Architecture

Project Overview

This project involved researching the G-share branch predictor and developing an optimized implementation in C++ to minimize mispredictions. The predictor was integrated into Gem5 simulation platform and evaluated across multiple benchmarks to analyze performance under different branch predictor table sizes.

The project focused on achieving significant reduction in misprediction rates through optimized parameters and comprehensive performance analysis across various configurations.

Key Achievements

  • Researched G-share branch predictor and developed optimized C++ implementation
  • Integrated predictor into Gem5 simulation platform
  • Ran three benchmarks: qsort, Dijkstra-small, and Dijkstra-large
  • Achieved 10% reduction in misprediction rate with optimized parameters
  • Analyzed predictor performance under different branch predictor table sizes
  • Delivered comprehensive report analyzing performance across multiple configurations
