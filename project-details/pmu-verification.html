PMU Verification Using UVM

Western Semiconductor
Jul 2025 - Present
Tempe, Arizona
SystemVerilog UVM Verification Code Coverage Functional Verification TLM

Project Overview

This project involves developing a comprehensive UVM-based verification environment for a Power Management Unit (PMU) that supports four distinct operational modes: Standby, Shutdown, Mission, and Startup. The PMU is a critical component responsible for managing power distribution and consumption across different system states.

The verification environment is designed to ensure the PMU operates correctly across all power modes and handles transitions between them seamlessly, maintaining system reliability and power efficiency.

Technical Implementation

UVM Architecture

The verification environment follows UVM best practices with a modular architecture:

  • Reusable Agents: Developed modular agents for different PMU interfaces including control signals, power rails, and status indicators
  • Layered Sequences: Implemented hierarchical sequences for different verification scenarios and power mode transitions
  • Configurable Stimulus: Created flexible stimulus generation that can be easily adapted for different test scenarios
  • Scoreboard: Implemented comprehensive scoreboard for functional verification and power mode validation

Power Mode Verification

Each power mode has specific verification requirements:

  • Standby Mode: Low power consumption with minimal functionality
  • Shutdown Mode: Complete power down with proper state retention
  • Mission Mode: Full functionality with optimal power consumption
  • Startup Mode: Controlled power-up sequence with proper initialization

Constraint Development

Developed mode-specific constraints to validate functional correctness:

  • Power rail voltage level constraints for each mode
  • Timing constraints for mode transitions
  • Current consumption limits and validation
  • State machine transition verification

Key Achievements

  • Achieved >95% code coverage across all PMU modules
  • Attained >95% toggle coverage ensuring all signal transitions are tested
  • Reached >95% condition coverage validating all conditional branches
  • Achieved >95% FSM coverage for complete state machine verification
  • Implemented directed test methodology for comprehensive functional validation

Impact & Results

This verification environment has significantly improved the reliability and quality of the PMU design:

  • Reduced post-silicon debug time through comprehensive pre-silicon verification
  • Ensured robust power management across all system states
  • Provided confidence in PMU functionality for production deployment
  • Established verification methodology standards for future PMU projects
