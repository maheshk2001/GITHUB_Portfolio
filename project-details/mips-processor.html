MIPS 5-Stage Pipelined Processor with Hazard Detection and Resolution

Arizona State University
Sep 2024 - Nov 2024
Tempe, Arizona
SystemVerilog Xilinx Vivado Computer Architecture RTL Design

Project Overview

This project involved designing a 5-stage pipelined MIPS processor implementing IF, ID, EX, MEM, and WB stages with comprehensive hazard detection and resolution capabilities. The processor includes a Hazard Detection Unit (HDU) and Hazard Resolution Unit (HRU) to handle RAW, WAR, and RAR hazards effectively.

The design incorporates a GShare Branch predictor using Global History and Program counter Indexing to minimize branch mispredictions and improve overall processor performance.

Key Achievements

  • Designed 5-stage pipelined MIPS processor with IF, ID, EX, MEM, and WB stages
  • Developed Hazard Detection Unit (HDU) and Hazard Resolution Unit (HRU)
  • Implemented handling for RAW, WAR, and RAR hazards
  • Designed GShare Branch predictor using Global History and Program counter Indexing
  • Verified pipeline execution using System Verilog testbenches
  • Achieved accurate instruction flow and stall minimization
