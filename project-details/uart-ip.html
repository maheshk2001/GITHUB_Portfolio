Full Duplex UART IP with Dual Clock FIFO Buffers

Arizona State University
Jun 2024 - Aug 2024
Tempe, Arizona
SystemVerilog Xilinx Vivado CDC RTL Design

Project Overview

This project involved designing an 8/16/32 bit UART interface with programmable baud rates between 960KHz to 3.2MHz. The design integrated asynchronous FIFO buffers for efficient TX and RX data management and clock domain crossing, ensuring reliable data transmission across different clock domains.

The UART implementation includes modular RTL for UART Controller and Tx/Rx paths with comprehensive logic for baud rate timing, start/stop signaling, flow control, and interrupt handling.

Key Achievements

  • Designed 8/16/32 bit UART interface with programmable baud rates (960KHz to 3.2MHz)
  • Integrated asynchronous FIFO buffers for TX and RX data management
  • Implemented clock domain crossing (CDC) for reliable data transfer
  • Developed modular RTL for UART Controller and Tx/Rx paths
  • Implemented baud rate timing, start/stop signaling, and flow control logic
  • Added comprehensive interrupt handling mechanisms
  • Verified UART operation under variable baud rates using System Verilog testbenches
